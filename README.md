# MemeGravity (Memeby)

这是一个基于驱动-行为-结果的中介模型开发的 H5 内容策略诊断工具。

## 项目特点

- **沉浸式设计**: 仿抖音风格，极简美学
- **动态交互**: 全程平滑过渡与微动画
- **专业诊断**: 基于理论路径验证
- **即时预览**: 实时雷达图与散点图可视化的分析结果

## 在线预览
